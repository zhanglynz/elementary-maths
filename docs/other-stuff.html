5.1 Interesting Number Series

5.1.1 Three integer sequences

We consider the following three integer sequences

Sequence 1: \(1, 2, 3, \ldots, n, \ldots,\)

Sequence 2: \(1^2, 2^2, 3^2, \ldots, n^2, \ldots,\)

Sequence 3: \(1^3, 2^3, 3^3, \ldots, n^3, \ldots,\)

Let \(S_n^{(k)}\) be the sum of the first \(n\) terms of the \(k\)th sequence in the above. Then, \[\begin{align} S_n^{(1)}&=\frac{n(n+1)}{2}, \tag{5.1}\\ S_n^{(2)}&=\frac{n(n+1)(2n+1)}{6}, \tag{5.2}\\ S_n^{(3)}&= \left(S_n^{(1)}\right)^2. \tag{5.3}\\ \end{align}\]

Proof of (5.1). We have \[2S_n^{(1)}=[1+2+\cdots+(n-1)+n] +[n+(n-1)+\cdots+2+1]=n(n+1).\] Thus, \[S_n^{(1)}=\frac{n(n+1)}{2}.\]

Proof of (5.2). We use mathematical induction method. \[ S_1^{(2)}=1^2=\frac{1\times (1+1)\times (2\times 1+1)}{6}, \] so, (5.2) holds when \(n=1\). Suppose that (5.2) holds when \(n=m\), where \(m\ge 1\), and now we show that (5.2) holds when \(n=m+1\). We have \[\begin{align} S_{m+1}^{(2)}&=S_m^{(2)} + (m+1)^2\notag\\ &=\frac{m(m+1)(2m+1)}{6} + (m+1)^2\notag\\ &=\frac{(m+1)[m(2m+1)+6(m+1)]}{6}\notag\\ &=\frac{(m+1)(2m^2+7m+6)}{6}\notag\\ &=\frac{(m+1)(m+2)(2m+3)}{6},\notag \end{align}\]

which completes the proof.

Proof of (5.3). We leave this proof for readers.

5.2 Serious Inequalities

5.2.1 Average is Between the Min and Max

Let \(x_1,\ x_2,\ \ldots,\ x_n\) be \(n\) numbers. Let \(w_1,\ w_2,\ \ldots,\ w_n\) be \(n\) positive numbers such that \(\sum_{i=1}^n w_i = 1\) (i.e. \(w_i\) for \(i=1, \ldots, n\) are the so-called weights, and a special case is \(w_1=w_2=\ldots=w_n=1/n\)). Let the minimum be \[ m:=\hbox{min}\{x_1, x_2, \ldots, x_n\}; \] let the maximum be \[ M:=\hbox{max}\{x_1, x_2, \ldots, x_n\}, \] The following are true.

  1. \(m \le x_i \le M\), for \(i=1, 2, \ldots, n.\)
  2. \(m\le \sum_{i=1}^n w_i x_i \le M.\)

Statement a is obvious; now we show b is true.

Proof of b. Since \[ m \le x_i, \ \hbox{for}\ i=1, 2, \ldots, n, \] \[ mw_i \le w_ix_i, \ \hbox{for}\ i=1, 2, \ldots, n. \] Thus, \[ m=\sum_{i=1}^n mw_i\le \sum_{i=1}^n w_ix_i. \] In the same manner, we can prove \[ \sum_{i=1}^n w_ix_i \le M. \]

Exercise (kind of challenging). Let \(x_1, x_2, \ldots, x_n\) be \(n\) arbitrary positive numbers; let \(y_1, y_2, \ldots, y_n\) be another \(n\) arbitrary positive numbers. Show that \[ \hbox{min}\{x_1/y_1, x_2/y_2, \ldots, x_n/y_n\}\le \frac{\sum_{i=1}^n x_i}{\sum_{i=1}^n y_i} \le \hbox{max}\{x_1/y_1, x_2/y_2, \ldots, x_n/y_n\}. \]

5.2.2 Cauchy-Schwarz inequality

Let \(\{a_1, \ldots, a_n\}\) and \(\{b_1, \ldots, b_n\}\) be two sets of arbitrary \(n\) real numbers, where \(n\) is a positive integer. Then \[\begin{equation} \left(\sum_{i=1}^n a_ib_i\right)^2\le \sum_{i=1}^n a_i^2\sum_{i=1}^n b_i^2. \tag{5.4} \end{equation}\]

Proof: Let \(\lambda\) be a symbol. It is always true that \[ \sum_{i=1}^n(a_i\lambda - b_i)^2\ge 0, \] that is, \[ \sum_{i=1}^n a_i^2\lambda^2-2\sum_{i=1}^n a_ib_i\lambda+\sum_{i=1}^n b_i^2\ge 0. \] If \(\sum_{i=1}^n a_i^2=0\), then (5.4) holds. Now, we assume \(\sum_{i=1}^n a_i^2\neq 0\), Then, the discriminant \[ \Delta=\left(-2\sum_{i=1}^n a_ib_i\right)^2-4\sum_{i=1}^n a_i^2\sum_{i=1}^n b_i^2\le 0, \] which implies that \[ \left(\sum_{i=1}^n a_ib_i\right)^2\le \sum_{i=1}^n a_i^2\sum_{i=1}^n b_i^2, \] and the equality holds if and only if \[ a_i\lambda-b_i=0,\ \text{for}\ i=1, \ldots, n, \] which in the language of linear algebra tells that the two vectors \((a_1, \ldots, a_n)\) and \((b_1, \ldots, b_n)\) are linearly dependent.

5.2.3 Relations among four means

Let \(x_1, x_2, \ldots, x_n\) be \(n\) positive real numbers. We can define

  • arithmetic mean: \(\mu_A=\sum_{i=1}^n x_i/n\)
  • harmonic mean: \(\mu_H=n/\sum_{i=1}^n x_i^{-1}\)
  • geometric mean: \(\mu_G=\left(\prod_{i=1}^n x_i\right)^{1/n}\)
  • square mean: \(\mu_S=\sqrt{\sum_{i=1}^n x_i^2/n}\)

The following holds \[ \mu_H\le \mu_G\le \mu_A \le \mu_S. \]

Proof of \(\mu_G\le \mu_A\). \[ \left(\prod_{i=1}^n x_i\right)^{1/n}\le \frac{\sum_{i=1}^n x_i}{n}\Leftrightarrow \frac{1}{n}\sum_{i=1}^n\ln x_i\le \ln\left(\frac{\sum_{i=1}^nx_i}{n}\right), \] which can be proved by using the fact that \(y=-\ln x\) is a convex function, and the Jensen Inequality: If \(f(x)\) is a convex function, then \[ f\left(\sum_{i=1}^n \lambda_i x_i\right) \le \sum_{i=1}^n \lambda_i f(x_i), \] where \(\sum_{i=1}^n \lambda_i=1\) and \(\lambda_i>0\) for \(i=1, \ldots, n.\)

Proof of \(\mu_H\le \mu_G\).

Let \(u_i=x_i^{-1},\ \text{for}\ i=1, \ldots, n.\) Then, \(\mu_H\le \mu_G\) becomes \[ \frac{n}{\sum_{i=1}^n u_i}\le \frac{1}{\left(\prod_{i=1}^n u_i\right)^{1/n}}, \] which holds because as we just proved \[ \left(\prod_{i=1}^n u_i\right)^{1/n}\le \frac{\sum_{i=1}^n u_i}{n} \]

Proof of \(\mu_A\le \mu_S\).

Taking square on both sides of \(\mu_A\le \mu_S\), we have \[ \left(\frac{\sum_{i=1}^n x_i}{n}\right)^2\le \frac{\sum_{i=1}^n x_i^2}{n}, \] which can be simplified as \[ \left(\sum_{i=1}^n x_i\right)^2 \le n\sum_{i=1}^n x_i^2. \]

According to the Cauchy-Schwarz inequality, we have \[ \left(\sum_{i=1}^n x_i\right)^2=\left(\sum_{i=1}^n x_i\cdot 1\right)^2\le \sum_{i=1}^n x_i^2\sum_{i=1}^n 1^2=n\sum_{i=1}^n x_i^2, \] which completes the proof.

5.3 Hard Combinatorics

5.3.1 Combination and permutation

We must distinguish two concepts, combination and permutation. In mathematics, a combination simply means a group—what matters is what members are in the group but not the order; whereas a permutation means a group—both membership and order matter. To be clear, let’s have an example.

Example: Let set \(\Omega=\{{\rm a, b, c, d}\}\). We select two different letters from \(\Omega\). List all the possible combinations and permutations from the selection.

Solution: The list of all the possible combinations is \[ \{{\rm ab, ac, ad, bc, bd, cd}\}. \] This list has 6 members.

The list of all the possible permutations is \[ \{{\rm ab, ba, ac, ca, ad, da, bc, cb, bd, db, cd, dc}\}, \] which has 12 members.

5.3.2 Permutation number \(P_n^k\)

Question 1: There are \(n\) balls and each of them has a unique number from \(1\) to \(n\). If we select \(k\) (\(1\le k \le n\)) different balls, how many permutations do we have?

Analysis: The above question is kind of abstract. Let’s look at a concrete example: We suppose \(n=4\) and \(k=3\). We can select balls with three steps; in each step we select one ball and write down the number of the ball. A table, which shows the idea, is as follows.

Step 1 Step 2 Step 3
1 12, 13, 14 123, 124, 132, 134, 142, 143
2 21, 23, 24 213, 214, 231, 234, 241, 243
3 31, 32, 34 312, 314, 321, 324, 341, 342
4 41, 42, 43 412, 413, 421, 423, 431, 432

From the above table, we can find there are \[4\times 3\times 2 = 24\ {\rm permutations}.\]

Answer to Question 1: Let \(P_n^k\) be the permutation number that we want to find out. Based on the above analysis, we can figure out that \[\begin{align} P_n^k&=n\times (n-1)\times (n-2)\times \cdots \times (n-k+1)\\ &=\frac{(n-1)\times (n-2)\times \cdots \times (n-k+1)\times (n-k)\times \cdots \times 1}{(n-k)\times \cdots \times 1}\\ &=\frac{n!}{(n-k)!k!} \end{align}\]

Remark: \(i!\)—read as \(i\)-factorial—is a shorthand notation for the product \(1\times 2\times \cdots\times i\).

5.3.3 Combination number \(C_n^k\)

Question 2: Same statement as that in Question 1, but we now ask: How many combinations do we have?

Analysis: Let’s again use the example in the Analysis for Question 1. The key is to find out how a combination is related to the permutations that contain the same members. For example, we have a combination \(123\); using the members in this combination, we can have all the permutations listed in the following table

Step 1 Step 2 Step 3
1 12, 13 123, 132
2 21, 23 213, 231
3 31, 32 312, 321

The above table tells us the combination \(123\) corresponds to \[ 3\times 2\times 1=3!=6\ {\rm permutations}. \] So, for \(n=4\) and \(k=3\) we want to find the unknown combination number \(x\), we have the equation \[ 3!x= 24, \] from which we get \[ x=\frac{24}{3!}=4. \]

Answer to Question 2: Let \(C_n^k\) be the combination number that we are seeking. Based on the analysis that we just did, we have \[ k!C_n^k=P_n^k, \] from which we get \[ C_n^k=\frac{P_n^k}{k!}=\frac{n!}{(n-k)!k!}. \]

Remark: In textbooks, the combination number is often denoted by \(\binom{n}{k}.\)

5.3.4 The binomial theorem

For any positive integer \(n\), \[\begin{equation} (a+b)^n = \sum_{k=0}^n C_n^k a^{n-k}b^k. \tag{5.5} \end{equation}\] Proof: The key is the equation \[ C_m^{k-1} + C_m^k=C_{m+1}^k, \] which can be easily proved. We use the mathematical induction method to prove (5.5). It’s easy to show that when \(n=1\) and \(n=2\) Equation (5.5) holds. Suppose that for \(n=m\) we have \[ (a+b)^m = \sum_{k=0}^m C_m^k a^{m-k}b^k. \] It follow that \[\begin{align} (a+b)^{m+1}&=(a+b)(a+b)^m\\ &=\sum_{k=0}^m C_m^k a^{m+1-k}b^k +\sum_{k=0}^m C_m^k a^{m-k}b^{k+1}\\ &=a^{m+1}+\sum_{k=1}^m C_m^k a^{m+1-k}b^k + \sum_{j=1}^{m+1} C_m^{j-1}a^{m+1-j}b^j\\ &=a^{m+1} +\sum_{k=1}^{m} (C_m^k + C_{m}^{k-1})a^{m+1-k}b^k + b^{m+1}\\ &=\sum_{k=0}^{m+1}C_{m+1}^k a^{m+1-k}b^k, \end{align}\]

which completes the proof.

5.3.5 Properties of \(C_n^k\)

Recall that a) the combination number \(C_n^k\) is the number of combinations when we select \(k\) different items from \(n\) items, and b) \[ C_n^k = \frac{n!}{(n-k)!k!],\ {\rm for}\ k=0,\1,\ \ldots,\ n, \] where \(0!=1\) by definition thus \(C_n^0=1\).

We now list some properties of \(C_n^k\)

  1. \(C_n^k = C_n^{n-k}\)

  2. \(C_n^k + C_n^{k+1}=C_{n+1}^{k+1}\)

  3. \(\sum_{k=0}^n C_n^k = 2^n\)

  4. \(\sum_{k=0}^r C_n^k C_m^{r-k}=C_{n+m}^r\)

  5. \(\sum_{k=0}^n C_n^k C_n^{n-k}=\sum_{k=0}^n\left(C_n^k\right)^2=C_{2n}^n\)

Proof: The first two equations are easy to be proved. For the third equation, noticing that \[ (1+x)^n = \sum_{k=0}^n C_n^k x^k, \] thus if we let \(x\) be 1 then we have the desired result. For equation 4, we notice that the RHS is coefficient of \(x^r\) in \((1+x)^{n+m}\), and the LHS is the coefficient of \(x^r\) in \((1+x)^n(1+x)^m\). Thus, LHS=RHS. Equation 5 is special case of equation 4, that is when \(n=m=r\).
